# Images Directory

This directory should contain the following image files required by the Tender Opulence Hub application:

- og-image.jpg - Open Graph image for social media sharing (1200x630px)
- twitter-image.jpg - Twitter card image (1200x600px)
- readme-banner.jpg - Banner image for the README.md file

## Image Requirements

### Social Media Images
- og-image.jpg: 1200x630 pixels (Facebook/LinkedIn optimal size)
- twitter-image.jpg: 1200x600 pixels (Twitter optimal size)
- Format: JPG or PNG
- File size: Optimize to under 1MB
- Content should include:
  - "Tender Opulence Hub" title
  - EdtoDo Technovations branding
  - Luxury visual elements (gold accents, marble texture)
  - Screenshot of interface (optional)

### README Banner
- Size: 1200x300 pixels (landscape orientation)
- Format: JPG or PNG
- Content should showcase the application's premium aesthetic

## Image Optimization

For production deployment, optimize images using:
- Image compression tools like TinyPNG 
- Proper image dimensions to avoid scaling
- Progressive loading for larger images
